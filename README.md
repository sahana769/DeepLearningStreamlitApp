# Fake vs Real Image Detection Web App

A **Deep Learning-based web application** built with **TensorFlow/Keras** and **Streamlit** to detect whether an image is **REAL** or **FAKE**.  
This project demonstrates practical skills in **image classification, model deployment, and Python web development**, suitable for showcasing in your portfolio or resume.

- Supports optional **preprocessing (`.pkl`)** and **configurable input shape/class names** (`model_config.json`).
